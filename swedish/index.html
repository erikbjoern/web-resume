Utbildning

Webbutveckling, fullstack

Craft Academy, Göteborgmars - juni 2020

14v intensivutbildning, starkt präglad av praktisk tillämpning. Jag fick väldigt goda möjligheter att inte bara utveckla mina färdigheter i att skriva kod, utan också i samarbete, kommunikation, att planera och organisera projekt, hitta information, felsöka mm. Jag mottog ett intyg på goda prestationer vid utbildningens slut. Kursens huvudinnehåll är:

  • Bygga front end- och back end-applikationer i främst React.js och Ruby on Rails
  • Tillämpa Scrum och andra agila arbetsmetoder såsom:
    • Testdriven utveckling (TDD) främst med Cypress och RSpec
    • Parprogrammering
    • Kontinuerlig integration och leverans/deployment (CI / CD)
  • Använda versionskontroll med Git och GitHub
  • Lära teori om bl.a. applikationsstruktur (The MVC pattern), HTTP och REST
